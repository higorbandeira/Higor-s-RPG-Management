# Princípios de IA

## Diretrizes essenciais
- Sempre siga os princípios da IA definidos neste módulo.
- Caso tenha dúvidas, pergunte antes de executar mudanças sensíveis.
- Priorize clareza, rastreabilidade e documentação em cada entrega.
- Separe decisões por módulo e evite misturar responsabilidades entre sistemas.
