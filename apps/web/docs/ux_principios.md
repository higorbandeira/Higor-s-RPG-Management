# Princípios de UX aplicados

## O que foi aplicado no layout
- **Hierarquia visual clara**: títulos maiores, subtítulos discretos e blocos separados por cards.
- **Espaçamento generoso**: padding e gaps consistentes para reduzir ruído visual.
- **Suavidade**: cantos arredondados, sombras leves e bordas sutis para destacar áreas interativas.
- **Consistência**: padrões de botões, inputs e cartões replicados nas telas.
- **Feedback imediato**: estados desabilitados/ocupados ficam mais opacos e legíveis.

## Dicas rápidas de UX para evoluir
- **Antecipar erros**: validação em tempo real para nickname e senha.
- **Guided flows**: textos curtos que explicam o próximo passo.
- **Estados vazios**: mensagens claras e ações recomendadas.
- **Acessibilidade**: contraste suficiente, foco visível, e labels explícitos.
- **Velocidade percebida**: skeletons ou loaders discretos para carregamentos.

## Bibliotecas recomendadas
- **UI/Componentes**: MUI, Chakra UI, Radix UI, Mantine.
- **Formulários**: React Hook Form + Zod (validação).
- **Data fetching**: React Query (TanStack Query).
- **Design system**: Tailwind CSS + Headless UI.
- **Animações**: Framer Motion (microinterações).
